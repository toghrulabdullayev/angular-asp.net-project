(1) Init commands:
      ng new client
      dotnet new webapi -n server -au None --no-https

(2) Angular CLI:
      ng generate c user -> generate a user component
      ng generate s shared/services/auth -> generate an auth service in shared/services/
      ng g p shared/pipes/firstKey -> generate a pipe in shared/pipes
      NOTE: "ng g" is the shorthand

      ng generate @angular/core:control-flow -> migrates from *ng directives to the @ ones
      ✔ Which path in your project should be migrated? ./src/app/user/login

(3) .NET intellisense config (mostly done automatically when opening a .NET project):
      dotnet new sln -> generates a solution file
      dotnet sln add YourProject.csproj -> adds .csproj to it

(4) .NET CLI:
      dotnet new gitignore
      dotnet add package Microsoft.AspNetCore.Identity.EntityFrameworkCore --version 8.0.26
      NOTE: version should be specified if the used .NET version is not the latest

(5) EntityFramework (EF) CLI:
      Installation and setup:
        dotnet tool install --global dotnet-ef
        dotnet add package Microsoft.EntityFrameworkCore.Design
        dotnet add package Microsoft.EntityFrameworkCore.Tools

      Migrations:
        dotnet ef migrations add InitialCreate
        dotnet ef database update
